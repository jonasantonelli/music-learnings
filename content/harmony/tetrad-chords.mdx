A practical chord dictionary for four-note voicings. Select a root note and the
string where you want the root, and see all common tetrad shapes at a glance.

Extended chords (7♭9, 7(9), m9, maj7♯11) use four notes — the 5th is dropped to
keep shapes playable while preserving each chord's characteristic sound.

### Chord families

- **Major** — maj7, 6, maj7♯11
- **Minor** — m7, m6, m7♭5, m(maj7), m9
- **Dominant** — 7, 7♯5, 7♭5, 7♭9, 7(9), 7sus4
- **Diminished** — dim7
