## Understanding ii-V-I voice leading

### Why ii-V-I?

The **ii-V-I** is the most common chord progression in jazz and popular harmony.
It appears in nearly every jazz standard and is the foundation for understanding
how chords move. Mastering drop 2 voicings through ii-V-I trains your ear and
fingers for real musical situations.

In a **major key** (e.g. key of C): Dm7 → G7 → Cmaj7 (ii-m7 → V-7 → I-maj7)

In a **minor key** (e.g. key of C minor): Dm7♭5 → G7 → Cm7 (ii-m7♭5 → V-7 → i-m7)

### Voice leading principles

Good voice leading means connecting chords with **minimal movement**. Instead of
jumping to a new shape across the neck, you find the nearest inversion of the
next chord. The goal:

- **Common tones stay** — if two chords share a note, keep it on the same string
- **Moving voices step** — other notes move by just 1-2 frets (a half or whole step)
- **No big jumps** — the hand barely moves between chord changes

This is what makes drop 2 voicings so powerful: each inversion of the ii chord
naturally leads to a nearby V voicing, which leads to a nearby I voicing.

### How to practice

- **Slow and deliberate** — play each path slowly, listening to how individual
  voices move from chord to chord
- **Identify common tones** — notice which notes stay on the same string between
  changes; these are your anchors
- **All four paths** — each starting inversion leads to a different voice leading
  path; learn all four to cover the full fretboard
- **All string sets** — shift each path to different string sets for a complete
  picture
- **Try all 12 keys** — once a path is comfortable in one key, cycle through all
  keys to build fluency
